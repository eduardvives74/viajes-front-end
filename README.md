# viajes-front-end

Prueba VueJs Front-end Develative

- Version NodeJS: 20.12.2
- Vue 3
- Vite
- Axios
- vue3-google-map (https://www.npmjs.com/package/vue3-google-map)
- sass-loader
- sass

## Como Ejecutar:

```sh
docker compose up -d
docker exec -it viajes_front_end /bin/bash
npm install
npm run dev
```

URL: http://localhost:8000/
